# PROMPT CLAUDE — WEBSITE & WEB PORTAL DEVELOPMENT

Buatkan sebuah website modern, professional, responsive, interactive, dan scalable dengan konsep:
“Website, Web Portal, dan Web Application untuk komunitas / residential cluster / RW digital”.

Nama Project:
Coatesville Website and Web Portal Development

## OBJECTIVE
Membangun website portal komunitas yang berfungsi sebagai:
- Portal informasi warga
- Media komunikasi cluster
- Portal pelayanan warga
- Portal kegiatan komunitas
- Portal laporan dan feedback
- Website modern dengan UI premium
- Responsive untuk desktop, tablet, dan mobile
- Memiliki CMS/Admin Panel
- Mendukung upload gambar, artikel, dan dokumentasi kegiatan
